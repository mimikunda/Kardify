# Kardify 
A lightweight, distraction-free flashcards app designed for local use and with modern design guidelines.

## Why?
The app serves as an intuitive and straightforward flashcards app with no ads or telemetry, allowing you to focus purely on the learning.
It is also developed as a learning experience for android development.

## Features
> [!IMPORTANT]
> The app is currently still in the early stages of development and has no working build.

## Tech stack 
- Jetpack Compose (M3E)
- Room (SQL)
